## Usage

```shell
$ npm install
$ echo 'JWT_KEY=$YOUR_JWT_KEY' > .env
$ createdb sweet_dev
$ npm run seed-dev
$ npm run dev
```

## Testing

```shell
$ createdb sweet_test
$ npm test
```

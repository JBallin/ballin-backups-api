# Sweet API

Frontend: [My Sweet Config](https://github.com/JBallin/sweet-config)
<br>
Deployed API: [sweet-api.herokuapp.com][1]

### Usage

```shell
$ npm install
$ echo 'JWT_KEY=$YOUR_JWT_KEY' > .env
$ createdb sweet_dev
$ npm run seed-dev
$ npm run dev
```

### Testing

```shell
$ createdb sweet_test
$ npm test
```

[1]: https://sweet-api.herokuapp.com
